# CoopStar Express - Landing Page

Landing Page moderna e responsiva para a CoopStar Express, focada em logística e frete rápido.

## 🚀 Tecnologias Utilizadas
- **HTML5 / CSS3**
- **Tailwind CSS** (via CDN)
- **Lucide Icons** (Ícones modernos)
- **AOS** (Animações de rolagem)
- **PHP** (Processamento de formulário)

## 📁 Estrutura do Projeto
- `index.html`: Página principal.
- `imagens/`: Assets e imagens geradas.
- `enviar_hotsite.php`: Script PHP para envio de leads por e-mail.
- `mapa.php`: Integração com Google Maps.

## ⚙️ Configuração
Para rodar o projeto localmente, basta abrir o `index.html` em qualquer navegador. Para que o envio de e-mail funcione, o projeto deve estar hospedado em um servidor com suporte a PHP.

### Variáveis de Ambiente
Renomeie o arquivo `.env.example` para `.env` (se estiver usando um sistema que suporte leitura de envs) ou utilize-o como guia para configurar os destinatários no script PHP.

## 📦 Publicação no GitHub
Para conectar este repositório local ao GitHub, utilize o script fornecido no arquivo `setup_remote.ps1`.
